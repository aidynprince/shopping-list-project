# Shopping List

A modern shopping list app built with vanilla JavaScript, CSS custom properties, and Vite.

## Features

- **Add items** with category, quantity, and auto-complete keyboard shortcuts
- **Edit & delete** items with smooth animations and undo support
- **Mark as purchased** with checkbox toggle
- **Filter/search** items with real-time text highlighting
- **Sort** by name, category, date added, or purchased status
- **Drag & drop** to reorder items
- **Dark/light theme** with system preference detection
- **Export/Import** list as JSON
- **Toast notifications** for all actions with undo capability
- **Keyboard shortcuts** (Ctrl+N new item, Ctrl+F focus search, Escape to cancel)
- **Persistent storage** via localStorage with legacy data migration
- **Fully responsive** mobile-first design

## Tech Stack

- [Vite](https://vitejs.dev/) — dev server & build tool
- Vanilla JavaScript (ES modules)
- CSS Custom Properties for theming
- HTML5 Drag & Drop API
- Font Awesome 6 icons

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
